# My Personal Portfolio 

A lightweight, two-page portfolio website designed to showcase my professional profile and creative projects using pure HTML and CSS.

##  Live Preview
Check out the live site here: [link]

##  Pages
- **Home/About:** A brief introduction, my skill set, and professional background.
- **Projects/Contact:** A dedicated space for my work gallery and ways to get in touch.

##  Built With
- **HTML5:** Semantic markup for better SEO and accessibility.
- **CSS3:** Custom styling focusing on a clean, modern aesthetic and responsive layout.

##  Project Goals
The goal of this project was to build a fast-loading, mobile-friendly personal brand site without relying on external libraries or frameworks.

##  How to View
Simply clone the repo and open `index.html` in any modern web browser.
